2.3 Product Management

On Profile Page, click to access Product Management view.

This is the page that agents use to view current products, and inventory levels as well as add any new products KH would like to begin selling.

The table contains a list of all producst including their ITEM CODE, DESCRIPTION, PRICE, VENDOR PRICE, QTY ON HAND, and an IMAGE associated with it. On the ITEM CODE header, there is a search bar that allows the user to search through items by the item code.

  • How to add a new product

    • On Product Management view, click on to display a dialog to add a new product.

    1. Please input all the information the form requests then click to save the new product into your inventory. As you add this product into your inventory, it will automatically appear on User Portal's Products Page.
      • ITEM CODE: Enter the item code for the product.
      • DESCRIPTION: Enter the description for the product.
      • PRICE: Enter the selling price for the product. *Positive number only.
      • VENDOR PRICE: Enter the vendor price for the product. *Positive number only.
      • QUANTITY: Enter the quantity on hand for the product. *Number only.
      • IMAGE: Upload an image for the product.*Image file size limited to 10Mb.
        • Note: Advised to use width x height for properly display on Products Page.
    2. If you want to cancel the current task of adding a new product, simply click the "x" at the top-right corner of the dialog to dismiss the dialog.
    3. If you are stuck on completing the task of adding a new product, click to seek instructions.

  • How to export Product table to Microsoft Excel

    • On the Product Management view, click on to export the table data to a CSV file format. Management team can open CSV file using Microsoft Excel to execute further analysis as needed.

2.4 User Management

On Profile Page, click to access User Management view.

This is the page that agents use to view and manage current users of the whole system including data for Admin as well as Agents.

The table contains a list of all Users of the system along with the Users' detail information including FIRST NAME, LAST NAME, USERNAME, BUSINESS NAME, REGION, and PHONE NUMBER. To view further details, there are two means of doing so:

  1. Management team can right click on a certain User then click . This will pop up MORE DETAILS dialog showing more detail information including: ADDRESS Detail, EMAIL, and ROLE.
  2. Management team can left click on a certain User to view all the User information.
Note: For a quick view of further details, use the first method.

Management team can filter the table by FIRST NAME, LAST NAME, USERNAME, BUSINESS NAME, and REGION using the Search box or Dropdown box under appropriate headers.

  • How to add a new User

    • On User Management view, click on to display a dialog to add a new user. This is to be used as Register a New User.

    1. Please input all the information the form requests then click to save the new user into your database.
      • EMAIL: Enter email address for the user. *Email format required.
      • USERNAME: Enter username for the user.
      • PASSWORD: Enter password for the user.
      • ROLE: Assign roles for the user.
        • Note: Admin can assign Agent and User roles. Agent can only assign User role.
      • FIRST NAME: Enter the user's first name.
      • LAST NAME: Enter the user's last name.
      • ADDRESS: Enter Street number and Street name of the user.
      • CITY: Enter the user's city.
      • STATE: Pick the user's state from the dropdown list.
      • ZIP: Enter the user's zipcode. *Number only.
      • REGION: Pick the user's region from the dropdown list.
        • Note: If there is no region specified, pick All regions.
      • PHONE NUMBER: Enter the user's phone number. *Number only.
      Note: EMAIL, USERNAME, PASSWORD, and ROLE are the only required fields. However, please be advised that all user's information should be entered.
    2. If you want to cancel the current task of adding a new user, simply click the "x" at the top-right corner of the dialog to dismiss the dialog.
    3. If you are stuck on completing the task of adding a new user, click to seek instructions.

  • How to edit a current User

  • How to delete a current User

  • How to export User table to Microsoft Excel

    • On the User Management view, click on to export the table data to a CSV file format. Management team can open CSV file using Microsoft Excel to execute further analysis as needed.

2.5 Vendor Management

On Profile Page, click to access Vendor Management view.

This is the page that agents use to place a replensishment order from the Vendor. The page contains 3 sections:

  • (1): Section where KH Agents actually add needed products to an order to be sent to the Vendor.
  • (2): Section where KH Agents can view the current products, and current inventory levels as well as add new products to this table to populate the new products in Section (1). This section functions like the Product Management view.
    • Refer to 2.2.3 Product Management
  • (3): Section where KH Agents can review the order before sending an invoice to the Vendor.
    • Note: (2) & (3) can be hidden.

  • How to place an order from the Vendor

    After reviewing, an invoice will be sent as an email to the Vendor upon clicking the

    • Click to display section (2), if not already visible. This section functions like Product Management view.
